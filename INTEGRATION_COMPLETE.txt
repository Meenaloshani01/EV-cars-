Algorithm:     ElasticNet Regression
R² Score:      0.875 (87.5% accuracy)
RMSE:          36.7 km
MAE:           ~28 km
Training Data: 25,000 records

Input Features (12):
  1. Car type          7. Road type
  2. Battery capacity  8. Speed
  3. Max range         9. Charging status
  4. Vehicle weight   10. Maintenance status
  5. Motor power      11. Total km run
  6. Battery %        12. Passenger count

Supported Vehicles:
  • Tata Punch EV (35 kWh, 315 km range)
  • MG ZS EV (50.3 kWh, 419 km range)
  • Volvo EX40 (69 kWh, 438 km range)

================================================================================
                         🔌 API ENDPOINTS
================================================================================

Base URL: http://localhost:5000/api

1. GET  /health           - Check API and model status
2. POST /predict          - Predict range for one vehicle
3. POST /predict/batch    - Predict range for multiple vehicles
4. GET  /car-types        - Get available car types and specs

Example cURL:

  curl -X POST http://localhost:5000/api/predict \
    -H "Content-Type: application/json" \
    -d "{\"car_type\":\"Tata Punch EV\",\"battery_percentage\":80,\"road_type\":\"City\",\"speed_kmph\":60,\"total_km_run\":15000,\"passenger_count\":4}"

                    💻 JAVASCRIPT INTEGRATION
================================================================================

Import the service:

  import { 
    predictRange, 
    checkMLHealth,
    getPredictedRange 
  } from './js/ml-service.js';

Get a prediction:

  const result = await predictRange({
    car_type: 'Tata Punch EV',
    battery_percentage: 80,
    road_type: 'City',
    speed_kmph: 60,
    total_km_run: 15000,
    passenger_count: 4
  });

  console.log(`Range: ${result.predictedRange} km`);

Check ML status:

  const health = await checkMLHealth();
  console.log('ML Available:', health.available);

================================================================================
                     🐛 TROUBLESHOOTING
================================================================================

PROBLEM: Model not loaded
SOLUTION: python backend/train_model.py

PROBLEM: CORS errors in browser
SOLUTION: pip install flask-cors

PROBLEM: Port 5000 already in use
SOLUTION: Change port in backend/app.py and js/ml-service.js

PROBLEM: Import errors
SOLUTION: pip install -r backend/requirements.txt

PROBLEM: API connection refused
SOLUTION: Make sure Flask is running: python backend/app.py
